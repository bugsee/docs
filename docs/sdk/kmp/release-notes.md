## 0.1.0 (April 22, 2026)
- Initial beta release of the Bugsee KMP SDK
- Unified Kotlin API wrapping native Bugsee Android SDK [6.0.2](/sdk/android/release-notes/#602-march-11-2026) and iOS SDK [6.1.3](/sdk/ios/release-notes/#613-april-21-2026)
- Targets: Android, iOS (arm64, x64, simulatorArm64)
- Published to Maven Central as `com.bugsee:bugsee-kotlin-multiplatform:0.1.0`
- Full feature set: video capture, crash reporting, network monitoring, console logs, custom data, feedback, appearance customization, lifecycle events, privacy controls
